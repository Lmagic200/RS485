
#RS485 project item configure declare


1. RS485 总线要求：
    1）物理接口：9600 bps， N81(如果是modbus协议，默认为1个起始位，8个数据位，一个偶校验位，一个停止位)
    2）一个RS485总线要求最多只接入一个面板
    3）面板MAC地址默认为0x01


2. 项目配置RS485设备地址段
    1）默认配置

            MAC configure / protocol: user defined
    ------------------------------------------------------------
        dooya： start ～end：0xef ~ 0xfe (239 ~ 254), curtain

    ------------------------------------------------------------
        leorey: start ~ end: 0xcf ~ 0xde (207 ~ 222), fresh air
    ------------------------------------------------------------
